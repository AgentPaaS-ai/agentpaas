# Archive

This directory keeps planning and asset history that is no longer an active
source of truth.

- `prd-history/` contains superseded PRD prompts, rendered PRDs, and advisor
  drafts.
- `checkpoints/` contains historical planning checkpoints from the pre-build
  review loop.
- `landing-page-legacy/` contains unused generated landing-page assets,
  generation helpers, and old static document pages.

Active implementation agents should use the root PRD, execution plan, and
subtask decomposition unless an issue explicitly asks them to inspect archived
material.
